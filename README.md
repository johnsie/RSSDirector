# RSSDirector
.net RSS reader based on RealNews

## Features
- Single EXE file
- Similar to FeedReader
- No Database
- Schedule download images
- Limit image sizes on download
- Update All on an interval
- Update single feed on an interval
- Import OPML files
- Dark mode
